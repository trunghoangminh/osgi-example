# OSGi Example
## 1. OSGi Provider
- To export service for another `Consumer` to use

## 2. OSGi Consumer
- To get service from `Provider`

## 3. OSGi Framework
- Create OSGi Framework, install bundles and run application
- Use `Main.java` to run App
